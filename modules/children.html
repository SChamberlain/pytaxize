Children¶

class pytaxize.Children(ids)[source]¶

Children: Retrieve taxonomic children

Usage:

from pytaxize import Children

# ITIS
## one id
x = Children(179913)
x
x.ids
x.itis()

## many ids - with one invalid id
x = Children([179913, 174321, 9999999])
x
x.ids
res = x.itis()
res[179913]
res[174321]
res[9999999]
